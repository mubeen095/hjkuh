What Should Be The First Tag In Any Html Document

Ans. Option C)<html> is the Correct answer

The <html> tag is the root element of an HTML document and is the first tag that should appear in the document. It summarise the entire HTML structure and contains other essential elements like <head> and <body> where you can include metadata, titles, scripts, and the visible content of your web page.
