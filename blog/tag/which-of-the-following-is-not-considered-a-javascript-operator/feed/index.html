Which Of The Following Is Not Considered A Javascript Operator

Answer. Option B)this is the Correct Option.

this” is not considered a JavaScript operator. “this” is a special keyword in JavaScript that refers to the current context or object, depending on how it’s used. It doesn’t perform any operation on data like the other options (new, delete, typeof) can.
