Which Of The Following Is A Privilege In SQL Standard

Ans. Option D is the Correct Answer

All of the privileges mentioned (SELECT, INSERT, and UPDATE) are standard privileges in SQL that control different types of actions that users can perform on database objects. So, all of these privileges are part of SQL’s permission system.
