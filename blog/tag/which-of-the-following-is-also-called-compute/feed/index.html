    • Which Of The Following Is Also Called Compute

    Answer: “Compute” in the context of cloud computing refers to the resources and services that handle processing tasks, such as running applications, executing code, and performing calculations. The term “compute” encompasses the virtual machines, containers, and serverless platforms that execute the code and perform computations.

    Out of the provided options, “set of Virtual Machines instances ” and ” set of commodity servers ” are both referred to as “Compute” resources in cloud computing. Set of Virtual Machine Instances This refers to a group of virtual machines (VMs) that are created and managed together. Virtual machines are software-based instances that mimic physical computers. They can run different operating systems and applications on a single physical server, providing flexibility and scalability for computing tasks.

    Set of Commodity Servers Commodity servers are basic, standardized servers that are not highly specialized. A “set of commodity servers” refers to multiple low-cost servers that are used collectively to handle computing workloads. Clustering these servers can provide enhanced processing power and redundancy

    These resources provide the computing power needed to run applications and perform tasks in the cloud environment.

    So, the correct answer is a set of Virtual Machines.
