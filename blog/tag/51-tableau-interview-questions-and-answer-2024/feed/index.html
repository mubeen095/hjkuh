Tableau Interview Questions And Answer 2024

  1. Tableau:
    Ans. Tableau is a powerful and popular data visualization tool that allows users to create interactive and shareable dashboards. It simplifies the process of exploring and understanding data by providing a user-friendly interface for creating visualizations, charts, and dashboards.
  2. Different Data Connection Options in Tableau:
    Ans. Tableau supports various data connection options, including:
  • Excel and Text Files
  • Databases (like MySQL, SQL Server, Oracle)
  • Cloud Data Sources (like Google BigQuery, Amazon Redshift)
  • Web Data Connectors
  • Data Extracts

3.Creating a Calculated Field in Tableau:
Ans.To create a calculated field in Tableau:

  • Right-click in the Data pane.
  • Choose “Create Calculated Field.”
  • Enter a name for the calculated field.
  • Define the calculation using Tableau’s formula language.

4.Difference Between Dimension and Measure in Tableau:

Ans.

  • Dimension: Represents qualitative data. Typically categorical and used for segmentation.
  • Measure: Represents quantitative data. Involves numerical values and is used for analysis.

5.Creating a Dashboard in Tableau:

Ans.

  • Go to the “Dashboard” tab.
  • Drag sheets or objects onto the dashboard.
  • Arrange and format as needed.
  • Add interactivity with actions and filters.

6.Data Blending in Tableau:
Ans. Data blending is used when working with multiple data sources. It combines data from different sources on a common dimension.

7.Purpose of a Parameter in Tableau:

Ans.
A parameter in Tableau is a dynamic input that can be used to create interactive and flexible visualizations. It allows users to replace a constant value in calculations, filters, and reference lines with a variable.

8.Performing Data Aggregation in Tableau:

Ans.
Tableau automatically aggregates data when creating visualizations. You can also use aggregation functions (SUM, AVG, COUNT, etc.) in calculated fields to perform custom aggregations.

9.Different Types of Charts in Tableau:

Ans.
Tableau offers a wide range of charts, including bar charts, line charts, scatter plots, pie charts, maps, heatmaps, treemaps, and more.

10.Sharing Tableau Workbooks:

Ans.
You can share Tableau workbooks by publishing them to Tableau Server, Tableau Online, or exporting them as packaged Tableau workbook files (.twbx). Sharing allows others to interact with your visualizations.

11.Data Types Supported in Tableau:

Ans.
Tableau supports various data types, including text, date/time, numeric, boolean, and geographical data types.

12.Understanding Dimensions and Measures:

Ans.
Dimensions are categorical fields used to segment and categorize data, while measures are numeric fields that contain quantitative data.

13.‘Discrete’ and ‘Continuous’ in Tableau:

Ans.

  • Discrete: Separate, distinct values used for categorical data.
  • Continuous: Uninterrupted range of values used for quantitative data.

14.Filters in Tableau:

Ans.
Tableau offers various filters, including:

  • Quick Filters
  • Context Filters
  • Top N / Conditional Filters
  • Relative Date Filters

15.Calculating Profit Percentage for a Customer Segment:

Ans.
Use a calculated field to calculate profit percentage for each segment, then analyze the proportion of total profits associated with the corporate segment.

16.Different Joins in Tableau:
Ans.Tableau supports different types of joins, including INNER JOIN, LEFT JOIN, RIGHT JOIN, and OUTER JOIN, to combine data from multiple tables.

17.Difference Between Joining and Blending:

Ans.

  • Joining: Combines tables within the same data source.
  • Blending: Combines data from different data sources.

18.Difference Between Live Connection and Extract:

Ans.

  • Live Connection: Connects directly to the data source.
  • Extract: Creates a static snapshot of the data for improved performance.

19.Calculated Field in Tableau:

Ans.
A calculated field is a field created by the user to perform calculations on existing fields in the data source.

20.Displaying Top and Bottom Sales:

Ans.

  • Use the “Top” or “Bottom” N filter.
  • Create a calculated field for ranking.
  • Apply filters to limit the view to the top and bottom sales.
